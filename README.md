# Shisu Sushi — Luxury Restaurant Website
Link: [shisu-sushi.vercel.app](shisu-sushi.vercel.app)

Welcome to **Shisu Sushi**, a visually striking, modern restaurant site crafted to showcase the finest digital experiences for fine dining. This project is a **portfolio mockup**, designed and developed to demonstrate design and frontend skills, presenting a tangible sample of what your web presence _could_ be.

---

## Features

- **Sleek, Responsive Design:** Pixel-perfect layouts tailored for all devices.
- **Interactive UI:** Smooth navigation, modern UI patterns, and elegant transitions.
- **Reservations Page:** Designed for real-world restaurant booking.
- **Component-Based Architecture:** Built with [Next.js](https://nextjs.org/) and [React](https://react.dev/).
- **Modern Forms:** Validations powered by [React Hook Form](https://react-hook-form.com/) and [Zod](https://zod.dev/).
- **Carousel Gallery:** Visually rich image showcase, built with Embla Carousel.

---

## Tech Stack

- **Framework:** Next.js 14, React 18, TypeScript
- **Styling:** Tailwind CSS 3
- **Forms:** React Hook Form + Zod
- **UI Components:** Custom, reusable React components
- **Other:** Embla Carousel for image sliders

---

## Purpose

This project was created as a portfolio piece. The restaurant, menu, reservations system, and all content are **fictional**. The intent is to demonstrate web development quality, UI/UX design ability, and technical range to potential business clients and hiring teams.

---

## Getting Started

Clone the repository, install dependencies, and run the development server:

```bash
git clone https://github.com/corey-farr/shisu-sushi.git
cd shisu-sushi
npm install
npm run dev
```

Visit [`http://localhost:3000`](http://localhost:3000) to view the site locally.
